Google Cloud Student Sprint Program @ Google

  • 由Google所舉辦，是一項在3月到6月期間進行為期 10 週的線上課程及工作坊的計畫。
  • 這個計畫網羅40名來自台灣、中國與海外的學生，並以5~6人小組形式由在 Google Cloud 擔任雲端工程師的 Googler 導師 進行分組指導。
  • 我們獲得有關 Google Cloud Platform 的基礎知識並沉浸於 Google 的企業文化中。
  • 該計畫的最終目標是帶領學生通過 Google Cloud Associate Cloud Engineer (ACE) 的專業認證考試。
  • 在這項計畫結束後成功考取 「Associate Cloud Engineer」之專業認證。
